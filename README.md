### 安装方式

#### 离线安装
- 从本项目的dist文件夹下获取.hpi后缀的文件信息
- 进入jenkin系统管理-->管理插件--->高级--->找到上传插件-->点击上传你刚刚下载的.hpi后缀的文件
- 等待插件安装完成以后，请重启你的jenkins，这样插件才能生效
- 进入你的jenkins任务，就可以看到当前插件了
- 一定要将webHook的密码填写你在码云WEBHOOK的密码一致(很重要，千万别忘了)

#### 在线安装
- 进入jenkin系统管理-->管理插件--->可选插件
- 等待安装完成
- 进入你的jenkins任务，就可以看到当前插件了
- 一定要将webHook的密码填写你在码云WEBHOOK的密码一致(很重要，千万别忘了)
